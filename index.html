Latest Project

Tourism In SriLanka

This website is a user-friendly platform designed to enhance travel planning for tourists and locals. It allows users to create personalized trips with route suggestions, budget planning, and detailed information on destinations, accommodations, and events. The platform features real-time updates on local activities, a packing list manager, and a news section for tourism trends. Users can also access travel agents, tour guides, and crowdsourced travel tips. Built with modern technologies, it ensures a secure and efficient experience, promoting Sri Lanka's tourism with innovative tools and resources.

Pharmacy Portal

"Queensway Pharmacy" is an online platform providing a wide range of pharmaceutical products and health-related items in Sri Lanka. It features an easy-to-navigate interface with categories for medicines, wellness products, and personal care, along with a prescription upload option for convenient ordering. Customers can add items to a cart, view product details, and enjoy a streamlined shopping experience. The site also includes an admin panel for managing user profiles, settings, and content, ensuring efficient operations and enhanced customer support.

Story Books Reader

The "Story Books Reader" app is a mobile application designed for Sinhala storybook enthusiasts. It provides a secure platform where users can register, log in, and explore a diverse collection of books. Built using Kotlin and Firebase, the app features user-friendly navigation, real-time book access, and category management. Administrators can add, update, or remove books efficiently. Users enjoy a personalized experience with secure authentication and easy access to content. This app serves as a modern solution for digital reading and library management in the Sinhala language.
